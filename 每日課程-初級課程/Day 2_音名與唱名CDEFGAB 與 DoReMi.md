# Day 2 - 音名與唱名（CDEFGAB 與 DoReMi）

## 📚 教材內容

音名（CDEFGAB）是用來表示音高的名稱，對應的唱名（Do Re Mi Fa Sol La Ti）是用來唱出音階的方式：
| 音名 | 唱名 |
| ---- | ---- |
| C    | Do   |
| D    | Re   |
| E    | Mi   |
| F    | Fa   |
| G    | Sol  |
| A    | La   |
| B    | Ti   |

這是音樂學習的基礎知識。
